This webpage helps you to visualize simple geometrical shapes such as circle and rectangle.

Circle

The $area$ of a circle with radius $r$ is given by $\pi r^{2}$. Similarly, the $perimeter$ of a circle with radius $r$ is given by $2\pi r$. Now, let's visualize the circle using Javascript and HTML.

Rectangle

The $area$ of a rectangle with width $w$ and height $h$ is given by $wh$. Similarly, the $perimeter$ of a rectangle with with width $w$ and height $h$ is given by $2(w+h)$. Now, let's visualize the rectangle using Javascript and HTML.
